## 简介

标签生成器是一个批量生成服装标签PNG图片的Python应用程序。它通过读取CSV/Excel数据文件，将文本和JAN-13条形码叠加到模板图片上，生成打印就绪的PNG文件。本文档提供了系统化的故障排除指南，帮助用户快速定位和解决在安装、配置、使用过程中遇到的各种问题。

    """清理数据质量问题"""
    cleaned_data = []
    
    for record in data:
        cleaned_record = {}
        
        # 清理空白字符
        for key, value in record.items():
            if isinstance(value, str):
                cleaned_record[key] = value.strip()

## 常见陷阱与规避方法

### 配置陷阱

**陷阱1：坐标系统混淆**
- **问题**：PIL坐标原点在左上角，x向右，y向下
- **规避**：始终以模板实际尺寸为准进行坐标计算
- **检查**：验证坐标值在(0,0)到(模板宽度,模板高度)范围内

**陷阱2：锚点理解错误**
- **问题**：anchor参数含义不清晰
- **规避**：使用标准锚点值："lt"(左上)、"mm"(中心)、"rt"(右上)
- **检查**：确认锚点与预期位置一致

**陷阱3：字体路径问题**
- **问题**：相对路径导致跨平台兼容性问题
- **规避**：使用绝对路径或基于项目根目录的相对路径
- **检查**：验证字体文件路径在不同环境下都能正确解析

### 数据陷阱

**陷阱1：JAN码格式不正确**
- **问题**：12位和13位JAN码处理差异
- **规避**：确保JAN码为纯数字，长度为12或13位
- **检查**：验证校验位计算正确性

**陷阱2：CSV编码问题**
